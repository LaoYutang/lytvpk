# 收藏服务器

“收藏服务器”用于保存常用 L4D2 服务器，查看状态并快速连接。

## 添加服务器

点击“添加服务器”，填写：

- 名称：自己容易识别的名字。
- 地址：服务器地址和端口，例如 `127.0.0.1:27015`。
- 权重：用于排序，数值越大越靠前。

保存后，服务器会显示在列表中。

## 查看状态

列表会查询服务器状态，并显示：

- 服务器名称。
- 游戏模式。
- 当前地图。
- 在线人数。

点击刷新可以重新查询。双击服务器可以查看详情和玩家列表。

## 连接服务器

点击“连接”会启动 L4D2 并连接到对应服务器。

主侧边栏底部的启动按钮会记录最近连接的服务器，方便下次快速进入。

## 导入和导出

服务器配置可以导入或导出：

- 导出到剪贴板。
- 导出到文件。
- 从剪贴板导入。
- 从 JSON 文件导入。

导入时会跳过已存在的地址，只添加新的服务器。

## 面板高级控制

面板高级功能适合服务器管理者使用，可以更方便地查看多个服务器状态，并快速执行常用管理操作。

> 高级面板功能需要服务器端先部署面板，并在收藏服务器配置中填写面板地址和密码后才会生效。

> 面板需要使用作者的 [LaoYutang/l4d2-server-next](https://github.com/LaoYutang/l4d2-server-next) 项目搭建。配置完成后，服务器列表会显示“面板”标记，并可以打开面板详情。

面板详情支持：

- 查看在线玩家。
- 刷新面板状态。
- 重启服务器。
- 切换地图。
- 修改难度。
- 上传地图文件。
- 清空地图。
- 发送 RCON 指令。
- 在浏览器中打开面板。

这些操作会直接影响服务器，请在确认没有玩家受影响后再执行重启、清空地图等危险操作。

## 常见问题

### 服务器查询失败怎么办？

服务器查询失败可能是地址错误、服务器未开放查询端口，或本地网络无法访问。

检查项：

- 地址格式是否包含端口。
- 服务器是否在线。
- 防火墙是否允许查询。
- 是否能在游戏内直接连接。

### 面板控制失败怎么办？

面板高级控制需要正确的面板地址和密码。

如果失败，请检查：

- 面板地址是否能在浏览器打开。
- 密码是否正确。
- 面板接口是否可访问。
- 服务器配置中的面板地址是否包含协议，例如 `http://`。
