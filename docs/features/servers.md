# 收藏服务器

“收藏服务器”用于保存常用 L4D2 服务器，查看状态并快速连接。

## 添加服务器

点击“添加服务器”，填写：

- 名称：自己容易识别的名字。
- 地址：服务器地址和端口，例如 `127.0.0.1:27015`。
- 权重：用于排序，数值越大越靠前。

保存后，服务器会显示在列表中。

## 查看状态

列表会查询服务器状态，并显示：

- 服务器名称。
- 游戏模式。
- 当前地图。
- 在线人数。

点击刷新可以重新查询。双击服务器可以查看详情和玩家列表。

## 连接服务器

点击“连接”会启动 L4D2 并连接到对应服务器。

主侧边栏底部的启动按钮会记录最近连接的服务器，方便下次快速进入。

## 导入和导出

服务器配置可以导入或导出：

- 导出到剪贴板。
- 导出到文件。
- 从剪贴板导入。
- 从 JSON 文件导入。

导入时会跳过已存在的地址，只添加新的服务器。

## 面板高级控制

如果服务器配置了面板地址和密码，会显示“面板”标记，并可以打开面板详情。

面板详情支持：

- 查看在线玩家。
- 刷新面板状态。
- 重启服务器。
- 切换地图。
- 修改难度。
- 上传地图文件。
- 清空地图。
- 发送 RCON 指令。
- 在浏览器中打开面板。

这些操作会直接影响服务器，请在确认没有玩家受影响后再执行重启、清空地图等危险操作。
