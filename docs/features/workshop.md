# 创意工坊浏览

“创意工坊”页面用于在应用内浏览 Steam Workshop 内容，适合直接查找 Mod 和合集。

## 分类浏览

左侧分类按 L4D2 常见内容整理，包括：

- 幸存者。
- 感染者。
- 模式和战役。
- 武器。
- 物品。
- UI、音效、脚本、模型、纹理等其他资源。

点击分类后，列表会按对应标签重新加载。

## 搜索和排序

顶部搜索框可以输入关键词。排序支持：

- 热门趋势。
- 最新发布或更新。
- 高评分内容。

还可以在“物品”和“合集”之间切换。

## 详情页

点击一个工坊卡片会进入详情页。详情页通常会显示：

- 标题、作者和预览图。
- 点击、订阅、收藏、文件大小和更新时间。
- 标签和描述。
- 下载按钮。
- “稍后再看”按钮。
- 使用浏览器打开。

如果是合集，详情页会列出合集中的子物品，并允许下载单个子物品。

## 稍后再看

“稍后再看”用于暂存想之后再处理的工坊物品。

适合浏览时先收藏一批目标，再统一下载或查看详情。

## 描述翻译

工坊详情页支持描述翻译。翻译服务可以在“设置”中选择：

- 微软翻译。
- Yandex 翻译。
- 自定义 AI 翻译。

自定义 AI 需要填写 Base URL、API Key 和模型 ID。

## 浏览器跳转

“使用浏览器打开”可以跳转到镜像站或 Steam 官方工坊。跳转目标在“设置”的“工坊设置”中配置。
