# 导入与拖拽

LytVPK 支持通过上传按钮或拖拽导入文件。拖拽时可以把文件直接放到应用窗口里。

## 支持的内容

| 类型                  | 处理方式                                |
| --------------------- | --------------------------------------- |
| `.vpk`                | 复制到当前 `addons` 目录并刷新列表      |
| `.zip`、`.rar`、`.7z` | 解压其中的 VPK 文件到当前 `addons` 目录 |
| 文件夹                | 打包成 VPK 后放入当前 `addons` 目录     |
| `.mdmp`、`.dmp`       | 打开崩溃转储查看器                      |
| 图片、GIF、TGA        | 打开喷漆制作工具                        |

导入 VPK、压缩包或文件夹前，需要先选择 `addons` 目录。

## 使用上传按钮

点击 MOD 管理页的上传按钮，选择要导入的文件。这个方式适合从文件选择器里挑选一个或多个文件。

上传按钮会复用和拖拽相同的导入流程。

## 使用拖拽

把文件或文件夹拖进应用窗口即可开始处理。

拖拽时会显示导入进度面板，包括：

- 当前处理的文件。
- 复制、解压或打包阶段。
- 总进度。
- 成功和失败结果。

如果压缩包里有多个 VPK，应用会尽量并行解压，并在完成后刷新列表。

## 喷漆素材拖拽

拖入图片、GIF 或 TGA 时，不会安装到 `addons`，而是打开喷漆制作工具。你可以在工具里预览、调整尺寸和格式，再保存为 VTF/VMT。

## 常见问题

### 导入失败怎么办？

常见失败原因包括：

- 尚未选择 `addons` 目录。
- 文件格式不支持。
- 压缩包里没有可用的 VPK 文件。
- 目标目录没有写入权限。
- 游戏或其他程序正在占用目标文件。

失败后可以查看导入面板中的失败信息，再重新拖入或手动处理文件。

如果是压缩包导入失败，可以先手动解压，确认里面是否有可用的 `.vpk` 文件。
