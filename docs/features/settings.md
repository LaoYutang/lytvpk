# 设置

“设置”页面分为网络设置、界面设置和工坊设置。

## 网络设置

网络加速用于创意工坊图片和文件下载。

可选模式：

- 自动优选最佳 IP：应用自动选择推荐线路。
- 手动指定 IP：从候选列表选择，或填写自建加速 IP。

如果当前网络访问 Steam CDN 正常，可以保持默认；如果图片加载慢或下载慢，可以尝试开启。

## 界面设置

界面设置用于调整 MOD 管理页的使用习惯。

- 显示模式：列表或卡片。
- 框选模式：拖拽绘制选择框来批量选择文件。
- Ctrl+单击选择：按住 Ctrl 点选多个 Mod。
- 筛选布局：简洁模式节省空间，经典模式展开更多筛选项。

## 工坊数据

“开启工坊信息存储”会为工坊文件创建 `.meta` 信息，用于保存名称、作者等数据。

开启后可以使用 Mod 更新检测。更新检测会检查带有工坊信息的 Mod 是否存在新版本。

## 工坊浏览器跳转

“跳转目标”决定在工坊详情里点击“使用浏览器打开”时打开哪里：

- 镜像站 `l4d2ws.com`。
- Steam 官方工坊。

## 描述翻译

描述翻译用于创意工坊详情页。支持：

- 微软。
- Yandex。
- 自定义 AI。

使用自定义 AI 时，需要填写 Base URL、API Key 和模型 ID。
