# 关于与更新

“关于”页面展示应用信息、项目链接和版本更新入口。

## 项目信息

关于页会显示：

- 作者。
- 开源协议。
- GitHub 仓库。
- 问题反馈入口。
- 当前版本。

项目仓库地址为：

```text
https://github.com/LaoYutang/lytvpk
```

## 检查更新

点击“检查更新”后，应用会请求最新版本信息。

如果发现新版本，会显示：

- 当前版本。
- 最新版本。
- 更新内容。
- 下载镜像选择。

## 下载镜像

更新弹窗支持 GitHub 直连和镜像地址。可以刷新镜像延迟，也可以填写自定义镜像。

如果下载很慢或失败，可以换一个镜像再试。

## 更新注意事项

- 更新时不要手动删除正在运行的程序文件。
- 如果杀毒软件拦截下载或替换，请先确认来源再放行。
- 更新完成后应用会重启。
