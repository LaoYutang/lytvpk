# 下载与解析

“下载与解析”页面用于把创意工坊链接、合集链接或直链转换为下载任务。

## 解析工坊链接

在输入框中粘贴 Steam 创意工坊链接或工坊 ID，点击解析。

支持的常见输入包括：

```text
https://steamcommunity.com/sharedfiles/filedetails/?id=123456
123456
```

解析成功后会显示标题、作者、文件名、大小和预览图。

## 合集下载

如果输入的是合集，应用会按组展示主物品和子物品。

你可以选择：

- 下载单个文件。
- 下载本组。
- 全部下载。
- 复制下载链接。

不可下载的物品会显示为不可用。

## 直链下载

如果你已经有文件直链，也可以直接填入下载链接并创建任务。

直链下载适合处理不是来自工坊解析结果的 VPK 文件。

需要注意的是，优选 IP 只会加速来自 Steam 创意工坊物品的 Steam CDN 下载链接。普通网站、网盘或其他第三方直链不会因为填写到这里就自动获得加速。

## 下载方式

下载任务会尽量使用多线程分块下载来保证较高的下载速度。应用会根据文件大小自动选择线程数；如果目标服务器不支持分块下载，会自动回退到普通单线程下载。

对于创意工坊文件和来自工坊物品的 Steam CDN 直链，开启优选 IP 后，下载连接也会经过 IP 优选。

## 下载队列

下载任务会进入后台队列。队列中可以查看：

- 当前状态。
- 下载进度。
- 文件大小。
- 速度和错误信息。

失败或取消的任务可以重试。已完成任务可以清理。

## 优选 IP 加速

开启“优选 IP 加速”后，应用会为 Steam CDN 选择更合适的线路，用于提升创意工坊图片和工坊文件下载体验。

如果正在优选线路，下载按钮会等待优选完成后继续。网络较差时可以关闭加速，或改用手动固定 IP。

## 常见问题

### 工坊解析失败怎么办？

创意工坊解析依赖外部网络。可以尝试：

- 重新解析工坊链接或工坊 ID。
- 检查网络正常。
- 稍后重试。
- 提交问题反馈。

### 下载任务失败怎么办？

先查看下载队列里的错误信息。常见处理方式：

- 点重试。
- 检查优选 IP 是否开启。
- 复制直链到浏览器确认链接是否还能访问。
