# VPK 打包

“VPK 打包”用于把一个资源目录打包成 `.vpk` 文件。

## 作用

当你已经准备好 L4D2 资源目录时，可以用这个工具生成 VPK，方便放入 `addons` 使用。

源目录应该是 VPK 根目录，也就是 `materials`、`models`、`scripts` 等文件夹的父目录。

## 示例结构

```text
my_mod
├── materials
├── models
└── scripts
```

选择 `my_mod` 作为源目录，而不是选择里面的 `materials`。

## 使用步骤

1. 打开“工具箱”。
2. 点击“VPK 打包”。
3. 选择源目录。
4. 选择输出位置，或直接输出到当前 `addons` 目录。
5. 等待打包完成。

如果输出到 `addons`，完成后会刷新 MOD 列表。

## 拖拽文件夹打包

也可以把文件夹拖进应用窗口。拖入文件夹时，应用会尝试把它打包为 VPK 并放入当前 `addons` 目录。

## 注意事项

- 源目录层级要正确。
- 文件名建议只使用常见英文、数字、下划线或短横线。
- 打包前确认目录中没有无关的大文件。
