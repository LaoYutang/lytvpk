# 模型面数检测

“模型面数检测”用于扫描 Mod 中模型资源的顶点数和三角形数量。

## 作用

高面数模型可能导致游戏卡顿、加载慢或显存压力变大。这个工具可以帮助你找出资源占用较高的 Mod。

## 检测范围

工具会读取启用和创意工坊 Mod 内的模型，统计 LOD0 的：

- 顶点数。
- 三角形数量。

LOD0 通常是最高细节模型，因此能反映模型资源的大致复杂度。

## 使用步骤

1. 打开“工具箱”。
2. 点击“Mod 模型面数检测”。
3. 等待扫描完成。
4. 查看结果中面数较高的 Mod 和模型路径。

## 结果怎么用

如果某个角色、武器或感染者替换包特别卡，可以优先检查它的模型统计。

处理方式可以是：

- 禁用高面数 Mod。
- 替换为更轻量的版本。
- 只在单人或低人数场景启用。

## 注意事项

该功能标记为 Beta，统计结果适合辅助判断，不代表一定会导致卡顿。
