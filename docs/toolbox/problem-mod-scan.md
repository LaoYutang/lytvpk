# 问题 Mod 查找

“问题 Mod 查找”用于定位导致游戏崩溃、进不去地图、模型错误或其他异常的单个 Mod。

## 作用

它按二分法工作：每一轮只启用一半候选 Mod，让你进入游戏测试结果，再根据“问题是否还存在”继续缩小范围。

这样比手动一个个开关 Mod 更快。

## 使用步骤

1. 打开“工具箱”。
2. 点击“问题 Mod 查找”。
3. 开始查找后，应用会准备第一轮测试范围。
4. 点击启动 L4D2，进入游戏复现问题。
5. 回到应用，提交本轮结果。
6. 继续下一轮，直到候选范围缩小到很少。

## 每轮怎么判断

- 如果问题仍然出现，说明问题 Mod 可能在当前启用的半区。
- 如果问题没有出现，说明问题 Mod 可能在另一半。

按照实际测试结果选择即可。

## 恢复原状态

查找结束或中途退出时，可以恢复查找前的启用状态。建议不要在查找过程中手动移动或删除候选 Mod，以免恢复失败。

## 注意事项

- 开始前建议关闭游戏。
- 测试时尽量只验证一个问题，不要同时改变其他游戏设置。
- 如果多个 Mod 一起导致问题，二分法可能只能先找出其中一个。
