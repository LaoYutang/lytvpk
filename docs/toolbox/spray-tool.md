# 喷漆制作

“喷漆制作”用于把图片或动画素材转换为 L4D2 可用的 VTF/VMT 喷漆文件。

## 支持素材

可以导入：

- PNG。
- JPG / JPEG。
- WEBP。
- BMP。
- GIF。
- TGA。

拖入这些素材时，也会自动打开喷漆制作工具。

## 使用步骤

1. 打开“工具箱”。
2. 点击“喷漆制作”。
3. 选择素材或批量导入素材。
4. 在预览区查看效果。
5. 调整输出设置。
6. 保存 VTF 或 VMT。

## 预览和动画

导入 GIF 或多帧素材后，可以：

- 查看当前帧。
- 上一帧、下一帧切换。
- 播放或暂停预览。
- 选择导入动画的起始帧、结束帧和速度。

## 输出设置

常用设置包括：

- 喷漆名称。
- 宽度和高度。
- 是否缩放以适应。
- 是否生成 Mipmaps。
- 纹理格式。
- 采样方法。
- 是否使用抖动。
- DXT 压缩质量。

推荐先使用默认设置，确认游戏内效果后再调整格式和质量。

## VTF 和 VMT

- VTF 是实际纹理文件。
- VMT 是材质描述文件，用来告诉游戏如何使用 VTF。

通常两个文件都需要保存并放到正确目录。

## 注意事项

- 图片尺寸过大可能导致文件体积偏大。
- 透明图建议使用支持透明度的格式，例如 DXT5。
- Mipmaps 需要满足原版规则，不是所有尺寸和格式都能开启。
