# 崩溃转储查看器

“崩溃转储查看器”用于打开 `.mdmp` 或 `.dmp` 文件，查看崩溃相关信息。

## 作用

当游戏或相关程序生成崩溃转储文件时，可以用这个工具快速查看：

- 异常信息。
- 线程信息。
- 模块列表。
- 内存范围。
- 原始 stream 信息。

这些信息可以帮助判断崩溃是否和某个模块、插件或资源有关。

## 使用步骤

1. 打开“工具箱”。
2. 点击“崩溃转储查看器”。
3. 选择 `.mdmp` 或 `.dmp` 文件。
4. 查看解析结果。

## 拖拽打开

也可以把 `.mdmp` 或 `.dmp` 文件拖进应用窗口。应用会直接打开第一个转储文件。

如果一次拖入多个转储文件，通常只会打开第一个。

## 注意事项

- 解析结果是辅助信息，不一定能直接给出崩溃原因。
- 如果要反馈问题，可以附上转储文件和当时启用的 Mod 列表。
