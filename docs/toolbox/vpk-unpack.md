# VPK 解包

“VPK 解包”用于把一个 `.vpk` 文件解开，查看里面的资源目录。

## 作用

解包后可以看到 VPK 内部的文件结构，例如：

```text
materials
models
scripts
sound
missions
```

适合检查一个 Mod 修改了哪些内容。

## 使用步骤

1. 打开“工具箱”。
2. 点击“VPK 解包”。
3. 选择要解包的 `.vpk` 文件。
4. 选择输出目录。
5. 等待解包完成。

应用会按 VPK 内部目录结构输出到同名文件夹中。

## 从文件列表解包

在 MOD 管理页的单个文件操作菜单中，也可以直接对某个 VPK 执行解包。

## 注意事项

- 解包不会自动修改原 VPK。
- 输出目录里已有同名文件时，请注意覆盖风险。
- 解包后的文件夹如果要重新变成 VPK，可以使用 [VPK 打包](/toolbox/vpk-pack)。
