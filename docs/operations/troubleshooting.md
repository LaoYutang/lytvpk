# 常见问题

## 扫描不到 Mod

先确认选择的是 L4D2 的 `left4dead2/addons` 目录，而不是游戏根目录或 `workshop` 上级目录。

可以尝试：

- 点“刷新”重新扫描。
- 检查文件扩展名是否为 `.vpk`。
- 确认文件没有被系统隐藏或移动到其他目录。

## 导入失败

常见原因包括：

- 没有先选择 `addons` 目录。
- 文件格式不支持。
- 压缩包里没有 VPK。
- 文件正在被游戏或其他程序占用。
- 目标目录没有写入权限。

如果是压缩包导入失败，可以先手动解压，确认里面是否有可用的 `.vpk` 文件。

## 创意工坊加载慢或解析失败

创意工坊依赖外部网络。可以尝试：

- 开启优选 IP 加速。
- 切换到手动固定 IP。
- 检查网络是否能访问 Steam 或镜像站。
- 稍后重试。

## 下载任务失败

先查看下载队列里的错误信息。常见处理方式：

- 点重试。
- 关闭优选 IP 后重试。
- 重新解析工坊链接。
- 复制直链到浏览器确认链接是否还能访问。

## 服务器查询失败

服务器查询失败可能是地址错误、服务器未开放查询端口，或本地网络无法访问。

检查项：

- 地址格式是否包含端口。
- 服务器是否在线。
- 防火墙是否允许查询。
- 是否能在游戏内直接连接。

## 面板控制失败

面板高级控制需要正确的面板地址和密码。

如果失败，请检查：

- 面板地址是否能在浏览器打开。
- 密码是否正确。
- 面板接口是否可访问。
- 服务器配置中的面板地址是否包含协议，例如 `http://`。

## 更新失败

可以尝试：

- 更换下载镜像。
- 关闭应用后重新启动再检查更新。
- 检查是否有杀毒软件拦截。
- 手动下载新版覆盖安装。

## 误删或误移动文件

删除和移动是真实文件操作。执行前请确认选中文件和目标目录。

如果担心误操作，建议先使用“导出 ZIP”备份一批重要 Mod。
