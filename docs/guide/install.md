# 安装与运行

## 系统要求

LytVPK 是 Windows 桌面应用，建议在 Windows 10 或 Windows 11 上使用。

## 启动应用

下载发布版后，运行 `LytVPK MOD管理器.exe`。应用启动后会进入主界面，并尝试读取上次保存的配置。

首次启动通常需要手动选择 L4D2 的 `addons` 目录。

## 配置保存位置

应用配置保存在：

```text
%APPDATA%\LytVPK\config.json
```

这里会保存目录历史、服务器收藏、工坊设置、界面偏好等信息。

## 自动更新

应用会在启动时检查新版本。发现新版本后，会显示更新内容，并允许选择下载镜像。

如果更新失败，可以：

- 换一个镜像源。
- 检查网络是否能访问 GitHub 或镜像。
- 关闭正在运行的旧版本后重试。

## 注意事项

- 管理 Mod 前建议关闭游戏，避免文件正在被占用。
- 删除、移动和批量禁用会影响本地 Mod 文件，请先确认选中范围。
- 创意工坊、图片预览、服务器查询和自动更新都依赖网络环境。
