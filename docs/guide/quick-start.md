# 快速开始

本页按第一次使用的顺序介绍。正常情况下，只要选对 `addons` 目录，就可以开始管理 Mod。

## 1. 选择 addons 目录

点击左上方的“选择 addons 目录”，选择 L4D2 的插件目录。

常见位置类似：

```text
...\Steam\steamapps\common\Left 4 Dead 2\left4dead2\addons
```

选择后，LytVPK 会保存目录历史。之后可以通过目录下拉菜单快速切换。

## 2. 扫描 VPK

选择目录后，应用会扫描 `.vpk` 文件并显示到 MOD 管理页。扫描结果会包含：

- 文件名、大小和位置。
- 是否启用。
- 自动识别的内容标签。
- 地图、预览图、工坊信息等可识别内容。

如果你刚复制了新文件，可以点“刷新”重新扫描。

## 3. 启用或禁用 Mod

列表中的启用状态决定 Mod 是否会被游戏加载。

- 单个 Mod：点击文件行上的启用/禁用按钮。
- 多个 Mod：勾选多个文件后，在底部使用批量启用或批量禁用。
- 全部禁用：批量禁用菜单里可以禁用全部 Mod，也可以只禁用人物或武器类 Mod。

禁用通常通过移动或改状态来完成，不建议在游戏运行中反复操作大量 Mod。

## 4. 搜索和筛选

你可以用搜索框查文件名、标题或其他可识别信息，也可以用标签和位置筛选快速缩小范围。

常用筛选方式：

- 按位置：区分 addons、workshop 或 disabled 等来源。
- 按一级标签：地图、人物、武器、音频等大类。
- 按二级标签：具体角色、武器类型、地图信息等。
- 显示隐藏：查看被隐藏的文件。

## 5. 导入和拖拽文件

导入有两种方式：

- 点击上传按钮选择文件。
- 直接把文件拖进应用窗口。

拖拽可以处理 `.vpk`、`.zip`、`.rar`、`.7z`、文件夹、崩溃转储文件和喷漆素材。详细说明见 [导入与拖拽](/features/import-drag-drop)。

## 6. 下载创意工坊 Mod

进入“下载与解析”，粘贴 Steam 创意工坊链接或工坊 ID，然后点击解析。

解析后可以：

- 下载单个文件。
- 下载合集中的一组或全部可下载文件。
- 复制直链。
- 查看下载队列、取消、重试或清理已完成任务。

也可以进入“创意工坊”页面直接浏览和搜索。

## 7. 收藏服务器并连接

进入“收藏服务器”，添加服务器名称和地址，例如：

```text
127.0.0.1:27015
```

添加后可以刷新状态、查看玩家列表，并点击“连接”启动 L4D2 连接服务器。
